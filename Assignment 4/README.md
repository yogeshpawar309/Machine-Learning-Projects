🚀 **Completed a K-Nearest Neighbors (KNN) Machine Learning Project!** 🎯

I recently built a **Disease Prediction** model using the **K-Nearest Neighbors (KNN)** algorithm as part of my machine learning learning journey.

The objective of this project was to predict whether a patient is likely to have a disease based on health-related attributes such as age, blood glucose level, cholesterol, blood pressure, BMI, smoking status, and physical activity. Since the target variable was either **"Yes"** or **"No"**, this was a **binary classification** problem.

After preprocessing the data, encoding categorical features, and applying feature scaling, I trained multiple KNN models with different values of **k (3, 5, 7, and 11)**. The model with **k = 5** delivered the best balance between training and testing performance, making it the most reliable choice. This value helped reduce overfitting while maintaining strong prediction accuracy.

One interesting insight from the dataset was that health indicators such as **glucose levels, cholesterol, BMI, and blood pressure** showed a strong relationship with disease occurrence, highlighting the importance of preventive healthcare.

One surprising thing I learned was how significantly **feature scaling** impacts KNN performance. Since KNN relies on distance calculations, unscaled features can dominate the prediction process and reduce model accuracy. Proper preprocessing made a noticeable difference in the model's results.

This project strengthened my understanding of classification, model evaluation, hyperparameter tuning, and the importance of data preprocessing in machine learning.
